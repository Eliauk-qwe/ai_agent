## 🎯 场景：做一个“AI问答网站”（类似 ChatGPT）

---

## ✅ 方案一：只用「灵积（DashScope）」

👉 你是纯后端开发者，自己写代码

### 你要做的事：

1. 写后端（C++ / Java / Python）
2. 调用灵积API（比如通义千问）
3. 自己写：

   * 前端页面
   * 对话逻辑
   * 历史记录
   * 用户系统

### 代码示意（伪代码）：

```python
import dashscope

response = dashscope.Generation.call(
    model="qwen-turbo",
    prompt="你好"
)
print(response)
```

👉 本质：

> 你 = 全栈工程师 + AI接口调用者

---

## ✅ 方案二：用「百炼（Model Studio）」

👉 你用平台直接搭应用（不用从零写）

### 你做的事：

1. 打开百炼平台
2. 选择一个模板（比如：AI客服 / 知识问答）
3. 配置：

   * 选择模型（通义千问）
   * 上传知识库（PDF、笔记）
4. 点几下 → 应用就跑起来了

👉 甚至可以：

* 不写代码
* 直接生成一个网页或API

---

## 🔥 核心对比（这个最重要）

| 维度   | 灵积      | 百炼      |
| ---- | ------- | ------- |
| 开发方式 | 写代码     | 可视化/低代码 |
| 灵活性  | ⭐⭐⭐⭐⭐   | ⭐⭐⭐     |
| 上手难度 | 高       | 低       |
| 适合人群 | 后端/系统开发 | 做项目/产品  |
